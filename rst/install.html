Installation¶

Instructions¶

Quick installation instructions. odelab depends on several packages in order to function properly, apart from scipy and numpy:

  • PyTables, a package for reading and writing files in the HDF5 format.
  • python-progressbar, a package to display progress bars
  • Theano will be used in future versions.

You may install those packages with pip by running the following command

pip install -r path/to/odelab/requirements.txt

The only package which might be problematic to install is PyTables.

Test the installation¶

You may try to run the test suite of odelab. (Make sure that you have nose installed first.) To run the tests, go to the root directory of odelab and run:

nosetest tests
